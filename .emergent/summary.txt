**Key Technical Concepts**
-   **Speech-to-Action Pipeline:** Utilizes a sequence of Whisper (Speech-to-Text) and a prompted LLM (GPT-4o-mini) to parse natural language admin commands into structured, executable JSON actions.
-   **Hybrid User Roles:** An Influencer is now a regular User with special flags (, ). The login flow was updated to create/link a user account and return a standard JWT, unifying authentication across the platform.
-   **User-to-User Transactions:** The gift system introduces a mechanism for users to transfer assets (bids) to one another, using a non-sensitive, publicly shareable Customer Number as the primary identifier.
-   **Dual-Channel Notifications:** A notification system that triggers both a persistent in-app notification (stored in MongoDB, displayed via a bell icon) and a transient email notification (via Resend) from a single backend event.
